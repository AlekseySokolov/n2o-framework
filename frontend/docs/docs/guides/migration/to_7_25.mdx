## Изменения в сборке {#changes-build}

- Обновление версии Selenide c 6.6.0 на 6.14.0
- Произведен полный переход с JUnit 4 на JUnit 5.
- Механизм чтения из Groovy файлов был полностью удален.

## Изменения в XML API {#changes-xml-api}
- Прекращена поддержка схемы `object-3.0`.
- Прекращена поддержка схемы `application-1.0`.
- Прекращена поддержка deprecated атрибута `name` в `<form-param>`.
Вместо него необходимо использовать атрибут `id`.
- Упразднен атрибут Размер строк таблицы - `table-size`, регулировать размер строк необходимо с помощью стилей.
- Упразднен атрибут Наличие чекбоксов в первом столбце таблицы `checkboxes`. Этот функционал можно реализовать с
помощью `selection="checkbox"`
- Упразднен атрибут Срабатывание (включение/выключение) чекбокса при клике по строке `check-on-select` у таблицы.
Теперь `selection="checkbox"` если по клику на строку прописано какое-то действие в таблице, то по клику будет оно
выполняться, если нет, то по клику на строку будет срабатывать чекбокс строки.
