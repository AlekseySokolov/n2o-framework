### &lt;menu-item&gt; {#menu-item}

Компонент меню

**Атрибуты**

|Наименование         |Тип                    |Описание                       |Значение по умолчанию|
|---------------------|-----------------------|-------------------------------|---------------------|
|id                   |Строка                 |Идентификатор компонента меню  |                     |
|name                 |Строка                 |Наименование компонента меню   |                     |
|datasource           |Строка                 |Идентификатор источника данных |                     |
|icon                 |Строка                 |Иконка компонента меню         |                     |
|badge                |Число                  |Значок компонента меню         |                     |
|badge-color          |Строка                 |Цвет значка                    |                     |
|badge-shape          |rounded, circle, square|Форма значка                   |circle               |
|badge-position       |left, right            |Позиция значка                 |right                |
|badge-image          |Строка                 |Картинка в значке              |                     |
|badge-image-shape    |rounded, circle, square|Форма картинки в значке        |circle               |
|badge-image-position |left, right            |Позиция картинки в значке      |left                 |
|image                |Строка                 |Путь к изображению             |                     |
|image-shape          |square, circle, rounded|Форма изображения              |rounded              |

**Тело**

[Действие открытия страницы](../xml/menu#open-page), [Открыть страницу браузера](../xml/menu#a)

**Пример**

```xml
    <menu-item id="notif"
               name="Уведомления"
               icon="fa fa-bell"
               badge="2"
               badge-color="warning"/>
```

### &lt;open-page&gt; {#open-page}

Действие открытия страницы

**Атрибуты**

|Наименование|Тип      |Описание          |Значение по умолчанию|
|------------|---------|------------------|---------------------|
|page-id     |Ссылка   |Ссылка на страницу|                     |
|route       |Строка   |URL страницы      |                     |

**Пример**

```xml
<page page-id="ProtoPage" route="/proto"/>
```

### &lt;a&gt; {#a}

Открыть страницу браузера

**Атрибуты**

|Наименование|Тип                         |Описание                                                                                                                   |Значение по умолчанию|
|------------|----------------------------|---------------------------------------------------------------------------------------------------------------------------|---------------------|
|href        |Строка                      |Адрес URL                                                                                                                  |                     |
|target      |self, newWindow, application|Сценарий открытия ссылки: self - в текущей вкладке, newWindow - в новой вкладке, application - открыть страницу приложения |self                 |

**Пример**

```xml
<a label="N2O" href="https://n2oapp.net/demo/" target="newWindow"/>
```

### &lt;dropdown-menu&gt; {#dropdown-menu}

Выпадающее меню

**Атрибуты**

|Наименование|Тип                    |Описание                      |Значение по умолчанию|
|------------|-----------------------|------------------------------|---------------------|
|id          |Строка                 |Идентификатор выпадающего меню|                     |
|name        |Строка                 |Наименование выпадающего меню |                     |
|icon        |Строка                 |Иконка выпадающего меню       |                     |
|image       |Строка                 |Путь к изображению            |                     |
|image-shape |square, circle, rounded|Форма изображения             |rounded              |

**Тело**

[Компонент меню](../xml/menu#menu-item), [Разделитель компонентов выпадающего меню](../xml/menu#divider)

**Пример**

```xml
<dropdown-menu id="user"
               name="Виктория"
               image="/static/users/vika91.png"
               image-shape="circle" >
    <menu-item>...</menu-item>
    <devider/>
</dropdown-menu>
```
